# Complaint Management System

## 📌 Description
A Java Swing based application with MySQL database to manage complaints.

## 💻 Technologies Used
- Java (Swing)
- MySQL
- JDBC

## 🚀 Features
- User Login
- Add Complaint
- View Complaint
- Update Status
- Delete Complaint

## 🛠️ How to Run
1. Import project in IDE
2. Setup MySQL database
3. Update DB credentials
4. Run MainApp.java
